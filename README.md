# React Layout Tools

![Image](https://i.imgur.com/L8PpWbU.gif)

A component to help with page layout including:

  - Grid
  - Guides
  - Mouse Guide
  - Page Margins (Future Release)
  - Rulers (Future Release)

## Getting Started

Install dependencies:

```
$: npm install
```

Run demo:

```
$: npm run start
```
